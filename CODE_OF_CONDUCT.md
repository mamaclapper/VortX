# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version: be decent. Critique code, not people. No harassment, no doxxing, no gatekeeping. Piracy-adjacent flame wars get closed; what users stream with their own addons is their business and not a debate this repo hosts.

Report problems through GitHub's report features or a private vulnerability report if sensitive. The maintainer moderates issues, PRs, and Discussions.
